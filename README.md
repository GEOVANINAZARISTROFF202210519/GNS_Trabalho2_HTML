<section>
<h2><b>3. O que é Hypertext Markup Language (HTML)</b></h2>

<p>
HTML (Hypertext Markup Language) é uma linguagem de marcação<br>
utilizada para criar e estruturar páginas na internet.<br>
Ela define a organização do conteúdo de um site,<br>
como títulos, parágrafos, imagens, links e tabelas.<br><br>

O HTML funciona através de elementos chamados <i>tags</i>,<br>
que indicam ao navegador como o conteúdo deve ser exibido.<br>
Atualmente o HTML é uma das principais tecnologias da web,<br>
sendo utilizado junto com CSS e JavaScript no desenvolvimento de sites.
</p>
</section>

<hr>

<section>
<h2><b>4. Como e por que surgiu o HTML</b></h2>

<p>
O HTML foi criado no início da década de 1990<br>
pelo cientista da computação Tim Berners-Lee.<br>
Ele trabalhava no CERN e precisava de uma forma simples<br>
de compartilhar documentos entre pesquisadores.<br><br>

Para resolver esse problema, foi desenvolvido um sistema<br>
de documentos conectados por links,<br>
que posteriormente ficou conhecido como a World Wide Web.<br>
Assim surgiu o HTML, permitindo estruturar essas páginas<br>
e facilitar o acesso às informações pela internet.
</p>
</section>

<hr>

<section>
<h2><b>5. O que é XHTML</b></h2>

<p>
XHTML significa eXtensible Hypertext Markup Language.<br>
Ele é uma reformulação do HTML baseada nas regras do XML.<br><br>

Seu objetivo foi tornar o código mais organizado,<br>
padronizado e compatível com diferentes sistemas.<br>
No XHTML é necessário seguir regras mais rígidas,<br>
como escrever todas as tags corretamente<br>
e fechar todos os elementos utilizados no documento.
</p>
</section>

<hr>

<section>
<h2><b>6. Diferença entre XHTML e HTML</b></h2>

<p>
A principal diferença entre XHTML e HTML<br>
está na forma como o código deve ser escrito.<br><br>

O HTML possui uma estrutura mais flexível,<br>
permitindo algumas variações na escrita das tags.<br>
Já o XHTML exige uma estrutura mais rígida,<br>
onde todas as tags devem ser corretamente fechadas,<br>
escritas em letras minúsculas e organizadas de forma padronizada.
</p>
</section>

<hr>

<section>
<h2><b>7. O que é XML</b></h2>

<p>
XML significa eXtensible Markup Language.<br>
É uma linguagem de marcação utilizada para<br>
armazenar, organizar e transportar dados.<br><br>

Diferente do HTML, o XML não foi criado<br>
para mostrar informações em páginas web,<br>
mas sim para estruturar dados de maneira clara<br>
para que diferentes sistemas possam compartilhar informações.
</p>
</section>

<hr>

<section>
<h2><b>8. Diferença entre HTML e XML</b></h2>

<p>
A principal diferença entre HTML e XML<br>
está na finalidade de cada linguagem.<br><br>

O HTML é utilizado para estruturar e apresentar<br>
conteúdos em páginas da internet,<br>
definindo como as informações serão exibidas no navegador.<br><br>

O XML, por outro lado, é utilizado para organizar<br>
e transportar dados entre sistemas,<br>
permitindo a criação de tags personalizadas<br>
de acordo com a necessidade da aplicação.
</p>
</section>
